# Agent Memory

本目录仅供 AI 助手在切换会话时快速恢复项目上下文，**不是面向用户的文档**。

## 索引

| 主题 | 路径 | 说明 |
|------|------|------|
| Animate.css 集成 | [animate-style/](./animate-style/) | 库用法、MoonHome 预设、已接入页面 |

## 使用方式（给 AI）

1. 接到动画/交互相关任务时，先读 `animate-style/03-implementation-map.md`
2. 需要选动画名或调参时，读 `animate-style/02-moonhome-presets.md`
3. 需要库 API 细节时，读 `animate-style/01-library-basics.md`
